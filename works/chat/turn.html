Call Network Settings

Calls now use secure TURN credentials from Firebase. The Metered secret key stays in Firebase and is never stored in this browser.

Secure public setup
  1. Metered domain: teamchatnishad.metered.live
  2. Store the Metered secret in Firebase as METERED_API_KEY.
  3. Deploy the Firebase Function named getTurnCredentials.
  4. Click Test Secure TURN below while logged in.

This works the same from desktop browsers, mobile browsers, and the installed web app because every client calls the same secure endpoint.
